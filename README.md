# Basic Portfolio — Vercel Deploy

This repo contains `index.html` and `portfolio.css` (CSS). Quick steps to deploy to Vercel.


Option 1 — Deploy via Git/GitHub (recommended for continuous deploy):

- Initialize git, commit, push to a GitHub repo.
- Connect the GitHub repo in the Vercel dashboard and enable deployments.

Notes:
- Vercel serves `index.html` from the project root. We copied your `portfolio.html` to `index.html` so the site is served at `/`.
- No additional config is required for a static site.
